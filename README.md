# Ruboty::Ss

ruboty plugin for take a screenshot of website.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-ss'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-ss

## Usage

    @ruboty ss http://example.com/
